# PortusSophia Jekyll v1 Template Contract

## Shared invariants

Every adopting surface must preserve:

1. A navy sailcloth header.
2. A PortusSophia gold lower header boundary on every page.
3. The gold anchor as the sole primary-navigation trigger.
4. A minimum 3.25rem navigation touch target and 2.35rem visible anchor.
5. One navigation tree emitted once in the document.
6. A visible skip link and `main` landmark.
7. Canonical favicon references.
8. A three-row footer: refrain; identity and links; copyright and rights.
9. The official programme mark and navigation anchor as distinct symbols.
10. Local content and local data remain owned by the adopting site.

## Canonical versus local

Canonical:

- layouts;
- includes;
- shared shell CSS;
- navigation behavior;
- brand material treatment;
- accessibility structure.

Local:

- page content;
- navigation entries;
- footer links;
- surface title and role;
- declared page composition;
- documented overrides.

## Prohibited drift

- Replacing the anchor with the programme mark.
- Removing the gold header boundary from selected pages.
- Duplicating desktop and mobile navigation markup.
- Hard-coding one surface's footer links into the shared include.
- Calling a local site copy canonical after the shared source changes.
- Automatically overwriting local overrides without review.
