# PortusSophia™ Brand Assets v0.2

A repository-ready package containing the harmonized brand brief, corrected supporting SVG assets, canonical material studies, review records, and a favicon system derived from the supplied programme-mark raster source.

## Important identity boundary

The supplied PortusSophia programme mark remains the official mark.

The harbor study, triad, and three-icon family are supporting or explanatory assets. They do not replace the programme mark.

## Favicon boundary

The favicon uses the supplied transparent programme-mark raster inside a fixed navy badge and restrained grey rim.

This provides stable contrast on both light and dark browser chrome without changing the mark's colors.

The favicon SVG embeds the source raster. It is a faithful derivative, not an original vector master.
